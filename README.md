# Sideload Server

A server application that enables sideloading apps without requiring a computer connection.

## Overview

This project provides a simple HTTP server that allows users to sideload applications directly to their devices wirelessly, without needing a computer or USB connection.

## Features

- 📱 Wireless app distribution
- 🖥️ Web-based interface
- ⚡ Fast and lightweight
- 🔒 Local network support

## Getting Started

### Requirements

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/elcactus11/The-watch.git
cd The-watch
npm install
```

### Usage

```bash
npm start
```

The server will be available at `http://localhost:3000`
